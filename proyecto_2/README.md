# Monitor de Impacto Social Minero

Este proyecto es una herramienta interactiva para visualizar y analizar la superposición entre unidades mineras y localidades indígenas en el Perú. Permite calcular la población afectada dentro de radios de influencia configurables.

## Tecnologías

- **Streamlit**: Framework de interfaz de usuario.
- **Geopandas / Folium**: Análisis y visualización geoespacial.
- **Pandas**: Manipulación de datos.

## Ejecución Local

1. Instalar dependencias:
   ```bash
   pip install -r requirements.txt
   ```
2. Ejecutar la aplicación:
   ```bash
   streamlit run app.py
   ```
